Tower Interior by Veldin
http://nwvault.ign.com/View.php?view=Hakpaks.Detail&id=3123


This is a set of tiles that can be used to build a (round) tower interior. It
contains two sets of tiles, one darker and one lighter version. All tiles are
embedded as features. This comes with HiRes DDS and LoRes TGA. Loadscreens are
also included This is v1.1, more tiles are certain to be added later.
